Hva er HTML?

HTML står for HyperText Markup Language. Det er språket vi bruker for å lage strukturen på nettsider.

Med HTML kan vi lage overskrifter, avsnitt, bilder, lenker og mye mer. Med bruk av blant annet HTML kan alle slags nettsider lages
