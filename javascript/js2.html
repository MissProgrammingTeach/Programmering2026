Hvordan fungerer spillet?

Dette spillet bruker HTML Canvas - et element vi kan tegne på med JavaScript.

Viktige konsepter:

Canvas og getContext('2d')
Gir oss verktøy for å tegne former, bilder og tekst
requestAnimationFrame()
Kjører spilløkken ca. 60 ganger per sekund for jevn animasjon
addEventListener('keydown')
Lytter etter tastetrykk slik at vi kan styre spilleren
Kollisjonsdeteksjon
Sjekker om to objekter (kurv og eple) overlapper
